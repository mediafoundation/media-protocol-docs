## Media Protocol, Media Network, Media Foundation
First, let's make clear the differences between the various parts of the **Media Protocol**, some of which may confuse new users.

- [Media Protocol](./protocol-overview): A suite of persistent smart contracts forming a protocol that allows anyone to create unstoppable peer-to-peer marketplaces.

- [Media Network](https://www.media.network): Anchored in Media Protocol's trustless system. A decentralized web services marketplace, offering a peer-to-peer alternative to giants like Google Cloud & AWS.

- [Media Foundation](https://www.media.foundation/): This entity serves as the driving force behind the development of the Media Protocol, along with Media Network.
