The `ProviderOperationParams` interface defines the parameters required to perform operations on a provider.

## Properties

- **`marketplaceId`**: `number | string | bigint`
The ID of the marketplace where the provider operation is being performed.

- **`provider`**: `string`
The address of the provider.
