The `InitializeMarketplaceParams` interface defines the parameters required to initialize a marketplace.

## Properties

- **`requiredStake`**: `number`
The required stake amount to register as provider of the new marketplace.

- **`marketFeeTo`**: `string`
The address to receive market fees.

- **`marketFeeRate`**: `number`
The rate of the market fee. E.g., 10000 for 1%.
