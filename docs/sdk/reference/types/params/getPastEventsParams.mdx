The `GetPastEventParams` interface defines the parameters required to retrieve past events from a contract.

## Properties

- **`eventName`**: `string[] | string | undefined`
The name(s) of the event(s) to retrieve.

- **`fromBlock`**: `bigint`
The starting block number to retrieve events from.

- **`toBlock`**: `bigint`
The ending block number to retrieve events up to.
