The `DealOperationParams` interface defines the parameters required to perform operations on a deal.

## Properties

- **`marketplaceId`**: `number | string | bigint`
The ID of the marketplace where the deal operation is being performed.

- **`dealId`**: `number | string | bigint`
The ID of the deal on which the operation is performed.
