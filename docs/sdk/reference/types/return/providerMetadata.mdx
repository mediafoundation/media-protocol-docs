The `ProviderMetadata` interface holds metadata about providers.

## Properties

- **`provider`**: `Address`
The address of the provider.

- **`metadata`**: `string`
Metadata related to the provider.

- **`publicKey`**: `string`
The public key of the provider for secure communication.
